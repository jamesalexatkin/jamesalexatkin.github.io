Battleship Antarctica

Battleship Antarctica is a 2D arcade-style shooter game written in Java. Players control battleships fighting amongst the icecaps of Antarctica, aiming to be the last one standing. Pickups are available to alter gameplay in various ways.

The game was developed as part of a team project in my second year of uni. It features a custom built rendering and physics engine, procedural terrain generation using Perlin noise, and supports singleplayer and multiplayer modes.
